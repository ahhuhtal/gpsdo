# GPS discliplined oscillator

Contained in this repository are the source code and PCB design files for my GPS disciplined oscillator project.

The GPSDO uses the one pulse per second (PPS) output of a GPS receiver to continuously adjust an OSC5A2B02 oven-controller crystal oscillator to keep it phase-locked with GPS time.

This gives a stable (both short term and long) 10 MHz time base for measurement instruments.

An important design goal has been to keep the cost very low.

Some more information is available at https://kiedontaa.blogspot.com/2025/02/gps-disciplined-oscillator-part-2.html
